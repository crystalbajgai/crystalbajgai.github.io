Qbasic - Uppercase and Lowercase Letter Counter

CLS
INPUT "Enter a word: ", word$
UPPER = 0
LOWER = 0
FOR i = 1 TO LEN(word$)
IF UCASE$(MID$(word$, i, 1)) = MID$(word$, i, 1) THEN UPPER = UPPER + 1
IF LCASE$(MID$(word$, i, 1)) = MID$(word$, i, 1) THEN LOWER = LOWER + 1
NEXT i
PRINT "Uppercase letters: "; UPPER
PRINT "Lowercase letters: "; LOWER
END

Explanation

The CLS statement clears the screen.
The INPUT statement prompts the user to enter a word and stores it in the variable word$.
The variables UPPER and LOWER are initialized to 0. These variables will store the count of uppercase and lowercase letters, respectively.
The FOR loop iterates over each character of the word.
The UCASE$ and LCASE$ functions are used to convert a character to uppercase and lowercase, respectively.
The MID$ function is used to extract a single character from the word.
If the character is equal to its uppercase version, then the value of UPPER is incremented.
If the character is equal to its lowercase version, then the value of LOWER is incremented.
After the loop, the final count of uppercase and lowercase letters is displayed using the PRINT statement.
